# ITmatch - Сервис для поиска IT-коллег

## Описание проекта
ITmatch - это веб-приложение для поиска и взаимодействия с IT-специалистами. 
Аналог Tinder для профессиональных знакомств в IT-сфере.

## Функциональность
- Регистрация и авторизация пользователей
- Создание профиля с информацией о специализации и опыте
- Система лайков и мэтчей
- Чат между мэтчами
- Поиск по специализациям
